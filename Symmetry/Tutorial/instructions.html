Instructions

On Each image:
  1. If you perceive a symmetry:
    • First choose symmetry type (rotation or reflection);
    • If it is a rotation symmetry, label its rotation center (one click);
    • If it is a reflection symmetry, label the two ends of the reflection axis (two clicks).
  2. If you label anything by mistake: Clear Last Label.
  3. If you do not perceive any symmetries: SKIP.
  4. If you finish all symmetry labeling: Submit.
